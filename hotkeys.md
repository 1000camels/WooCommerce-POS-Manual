# HotKeys


Hotkeys are keyboard shortcuts which can be used to quickly trigger actions in WooCommerce POS.

| Key | Description | Version Introduced |
| -- | -- | -- |
| ? | Help screen |  0.4 |
| B | Barcode mode |  0.4 |
| V | Search mode |  0.4 |
| S | Sync with server |  0.4 |
|  | Checkout |  0.4.6 |
|  | Process Payment |  0.4.6 |
